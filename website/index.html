Search Engine Construction Kit

SECK is an open source Search Engine Construction Kit. Its goal is to provide end-users with the ability to build their own search engine from the ground up.

Design

SECK's design consists of three major subsystems:

Online Frontend/Backend

  • Query Interface

  • Pluggable and Extensible Ranking Engine

Offline Backend

  • Crawler

  • Index

  • Classification

Distributed/Cluster Configuration

  • MapReduce Computation Grid Engine

Description

The Search Engine Contruction Kit is a project started and facilitated by computer science professor John Tran at the California State University, Los Angeles. In conjunction with his computer science students, John aims to develop a fully functional search engine platform for others to use and contribute to. The project can be viewed and downloaded on GitHub.
